# Mobile App (React Native)

A React Native mobile application.

## Prerequisites

- Node.js 18+
- For iOS: Xcode and CocoaPods
- For Android: Android Studio and Android SDK

## Development

Start the Metro bundler:

```bash
pnpm dev
```

In a separate terminal, run on your target platform:

### iOS
```bash
pnpm ios
```

### Android
```bash
pnpm android
```
